# Hello World Examples

This repository contains "Hello World" examples in various programming languages.

## Running the Examples

### Python
```
python hello_world.py
```

### JavaScript
```
node hello_world.js
```

### Shell Script
```
./hello_world.sh
```

### C
First compile, then run:
```
gcc -o hello_world_c hello_world.c
./hello_world_c
```
